The OpenBook Project

About

This repo is the core infrastructure for The Open Book project. In here is the interperter and testing infrastructure.

Mission Statement

The goal is to provide a powerful yet friendly to use syntax that allows users to create high quality digital media for education.

Features

  • Simple markdown-like syntax but with more control
  • Generates responsive HTML that works across all devices
  • Accessible by default, HTML that is WAVE compliant

Contributing
